Sobre Nós

Nossa Missão

Este site foi criado como um projeto para a disciplina de Programação Web. Nossa missão é aplicar os conceitos aprendidos de HTML, CSS, JavaScript e PHP de forma prática, criando um catálogo de receitas que seja funcional e fácil de usar.

Mais do que apenas um trabalho acadêmico, o "Culinária Fácil" é uma forma de resgatar e compartilhar aquelas comidas afetivas que marcam nossas vidas, utilizando a tecnologia para conectar gerações.
